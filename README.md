# 🚀 QuantInsight-AI

AI-powered quantitative risk analysis platform combining technical indicators, real-time market data, and news-driven sentiment to generate explainable investment insights.

---

## 🔥 Features

- 📊 Real-time stock data (RSI, Moving Average)
- 🧠 AI-generated risk score & reasoning
- 📈 Multi-stock comparison dashboard
- 📰 News-based sentiment analysis
- 🎨 Interactive Streamlit UI

---

## 🛠 Tech Stack

- Python
- Streamlit
- yfinance
- Plotly
- Gemini API
- NewsAPI
